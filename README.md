# 🏦 MBank - Spring Boot Banking System

A modern, full-featured online banking system built with **Spring Boot 3**, **Thymeleaf**, and **MySQL**. Provides secure banking operations with a responsive web interface.


## ✨ Features

### 🔐 Authentication & Security
- **User Registration** with auto-generated account numbers
- **Secure Login/Logout** with Spring Session management
- **Session-based authentication** with automatic timeout
- **Form validation** with real-time feedback

### 💰 Banking Operations
- **Deposit Money** with amount validation and limits
- **Withdraw Funds** with automatic balance checking
- **Transfer Between Accounts** with transaction fee calculation
- **Real-time Balance Updates** after every transaction

### 📊 Account Management
- **Interactive Dashboard** with account summary
- **Transaction History** with filtering by date and type
- **Account Statements** with customizable date ranges
- **Transaction Receipts** with unique IDs
- **Export Functionality** (Print to PDF)

### 🎨 User Experience
- **Modern, Responsive UI** with gradient designs
- **South African Rand (ZAR)** currency formatting
- **Visual transaction flows** with color-coded types
- **Quick amount buttons** for common transactions
- **Mobile-friendly** interface

## 🛠️ Tech Stack

### Backend
- **Java 17** with **Spring Boot 3.2.5**
- **Spring MVC** for web layer
- **Spring Data JPA** for database operations
- **Spring Security** patterns (session-based)
- **Hibernate** ORM with automatic schema updates

### Frontend
- **Thymeleaf** template engine
- **HTML5, CSS3, JavaScript** with modern styling
- **Responsive Design** with CSS Grid/Flexbox
- **Font Awesome** icons
- **Client-side validation**

### Database
- **PostgreSQL** (Production on Render)
- **MySQL** (Local development)
- **JPA/Hibernate** for object-relational mapping
